#### 꼬리질문) Java가 제공하는 기본 함수형 인터페이스에는 어떤것들이 있나요?

<details>
    <summary>답변</summary>

- Predicate : 두 객체를 비교할 때 사용, boolean 값 반환
- Supplier : 제네릭으로 선언된 타입을 반환
- Consumer : 매개변수를 받을 수 있으며, 작업을 수행 후 반환하지 않음
- Function : 제네릭 T, R 으로 T 매개 타입, R 반환 타입

</details>
